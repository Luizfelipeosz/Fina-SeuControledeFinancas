# 💰 Fina — Controle de Finanças Pessoais

O **Fina** é uma aplicação de controle financeiro pessoal desenvolvida para auxiliar usuários na organização de receitas e despesas de forma simples, intuitiva e eficiente.

A ferramenta permite o registro de entradas e saídas financeiras, oferecendo uma visão clara do saldo e do histórico de movimentações, contribuindo para um melhor acompanhamento da vida financeira no dia a dia.

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com foco em:

- Organização e clareza na gestão financeira pessoal
- Aplicação de boas práticas de desenvolvimento Front-End
- Estruturação de código limpo e escalável
- Criação de uma experiência de usuário simples e funcional

Além disso, o projeto também representa um processo contínuo de aprendizado, onde foram aplicados conceitos de desenvolvimento web, lógica de programação e arquitetura básica de aplicações.

---

## 🧠 Principais Funcionalidades

- Cadastro de receitas e despesas
- Cálculo automático do saldo financeiro
- Visualização clara do histórico de transações
- Interface simples e intuitiva
- Organização por tipo de movimentação
