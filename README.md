# Fina-SeuControledeFinancas
O Fina  é uma aplicação para controle financeiro pessoal, desenvolvida para ajudar os usuários a organizarem suas receitas e despesas de forma simples e intuitiva. O projeto permite o registro de entradas e saídas financeiras, oferecendo uma visão clara do saldo e dos gastos, facilitando o acompanhamento da vida financeira no dia a dia.
